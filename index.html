What is semantic HTML?

Semantic HTML introduces meaning to the code we write. Before Semantic HTML the elements didn’t have any meaning as to what it does or what content goes in it. With Semantic HTML we were introduced to elements that tell developers and browsers exactly what it does and what content should go in it.(codeacademy).
